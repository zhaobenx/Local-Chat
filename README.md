# 一个局域网聊天软件|A local chat sofrtware
一个分布式的局域网聊天软件

A distributed P2P chat software used in LAN

## 目标|Target

* 实现用 UDP 广播来寻找客户端
* 利用 TCP 来进行文字以及文件的传输
* 写一个图形界面

## 进程|Progress

- [x] UDP 发现 | UDP discovery
- [ ] TCP 广播聊天 | TCP broadcast chat
- [ ] TCP 群组聊天 | TCP group chat
- [x] TCP 一对一聊天 | TCP private chat
- [ ] 文件传送 | Send files
- [ ] 图形界面 | GUI
